ABOUT OUR PROJECT...

Our project aims at decentralisation of file storage by using multiple servers enabled by the IPFS and blockchain technology

Project description

The data (in this case confidential documents) through the website and stored in IPFS networkusing python back-end server. The IPFS hash is obtained and a transaction is created to store IPFS hash in the blockchain.Ether is paid to process the transaction. Smart Contracts are coded and compiled using solidity language. These smart contracts are deployed using the transaction by paying the ethers.
If the transaction is successful the smart contract is stored on ehtereum virtual machine. This IPFS hash is also stored on the local blockchain. Here we us local ethereum blockchain called Ganache. Ganache runs on ethereum blockchain and provides a testing environment.

TOOLS USED

  1. Ethereum Blockchain
  2. IPFS network: Inter Planetary File System
  3. Ganache : Local ethereum blockchain node
  4. React JS: For dynamic frontend interaction
  5. Solidity: Smart Contracts language
  6. Django Framework: Python backend server
